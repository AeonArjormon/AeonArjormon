2024/4/17

1.采用反复查库的方法。

cursor.execute("select schemaname, tablename from pg_catalog.pg_tables order by schemaname")
# 获取查询结果
rows = cursor.fetchall()
flag = False
for db, sc in tqdm(rows):
    rows = cursor.fetchall()
    cursor.execute(f"SELECT pg_total_relation_size('{db}.{sc}')")
    print(f"SELECT pg_total_relation_size('{db}.{sc}')")
    size = cursor.fetchall()
    with open("./out.txt", "a") as f:
        f.write(f"INSERT INTO analyze_dataset VALUES ('urn:li:dataset:(urn:li:dataPlatform:postgres,{dbname}.{db}.
                {sc},PROD)', {rows[0][0]}, {size[0][0]});\n")

可以实现获取所有表的特定信息，但是由于数据量过大，统计时会耗费极长的时间，所以建议数据量不是特别大的时候采用。

2.优化上述方法，更换时实现方式。
